# matplotlib 使い方

## 基本的な考え方
```txt
+----------------------------+
|           figure           |
| +---------+    +---------+ |
| | subplot |    | subplot | |
| |         |    |         | |
| +---------+    +---------+ |
|                            |
+----------------------------+
```

1. `plt.figure(n)`
  + n番目の図を作成する場所を確保
2. `plt.subplot(タテ, ヨコ, m番目)`
  + タテ・ヨコの区切りの数を決め、そのm番目に描画することを宣言
3. `plt.plot()` etc...
  + 選択した領域に`plot`などする
  + `pd.DataFrame.plot()`なども利用することができる


## よく使うTips
### x軸, y軸の値域を設定する
`plt.xlim(min, max)`や`plt.ylim(min, max)`を利用することで設定できる。
例えば、`plt.xlim(0.0, 0.3)`などとすると、`x軸`の上限と下限が設定される。
